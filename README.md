# Modern Calculator

A sleek, responsive, and feature-rich calculator application built with vanilla HTML, CSS, and JavaScript. This project features a modern Neumorphic/Glassmorphism design with full support for light and dark modes.

##  Features

- **Modern UI**: Stylish design using CSS gradients, shadows, and backdrop filters.
- **Dark/Light Mode**: Toggle between themes with automatic persistence using `localStorage`.
- **Keyboard Support**: Full support for numeric keys, operators, Enter (`=`), Backspace (`DEL`), and Escape (`AC`).
- **Responsive Design**: Works seamlessly on desktops, tablets, and mobile devices.
- **Floating Point Precision**: Handles decimal calculations and rounds results to avoid common floating-point errors.
- **Interactive Animations**: Smooth transitions, button click effects, and loading animations.
- **Error Handling**: Visual feedback for invalid expressions (e.g., division by zero).

##  Getting Started

To run this project locally:

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.

##  Built With

- **HTML5**: Semantic structure for the calculator interface.
- **CSS3**: Advanced styling including Flexbox, Grid, CSS Variables, and Keyframe animations.
- **JavaScript (ES6+)**: Functional logic for calculations, theme management, and event handling.

##  Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9`, `.` | Input numbers and decimals |
| `+`, `-`, `*`, `/` | Arithmetic operators |
| `Enter` or `=` | Calculate result |
| `Backspace` | Delete last character |
| `Escape` | Clear display |

##  Project Structure

- `index.html`: The main entry point and UI structure.
- `style.css`: All styling, including theme variables and animations.
- `script.js`: Calculator logic, event listeners, and theme toggling.
